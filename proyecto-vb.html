Tipo :Aplicación de escritorio

Detalles del proyecto

Este proyecto surge de la necesidad de un centro de estética para informatizar el negocio y poder llevar de una manera más cómoda y eficaz la gestión y venta de productos y servicios así como la gestión del stock y un control de caja.

Características

La solución adoptada para cubrir la necesidad del cliente fue programar la aplicación en Visual Basic y Access para una pequeña base de datos.

Decidí hacer la aplicación en este lenguaje por varios motivos. El principal por el hecho de tener muy reciente este lenguaje por finalizar un curso de programación. Otro motivo fue por la facilidad y flexibilidad de crear la parte gráfica, algo que era bastante importante teniendo en cuenta que se trataba de una aplicación a medida, con cambios constantes a petición del cliente y una aplicación dirigida a usarla mediante pantalla táctil, cajón portamonedas e impresora de tickets.
