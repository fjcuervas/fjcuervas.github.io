Detalles del proyecto

Se trata de una aplicación web destinada a la gestión comercial de pequeñas empresas con el objetivo de poder llevar un control sobre la venta de productos y servicios.

Características

Para el desarrollo de la aplicación utilizo el estándar Java Enterprise Edition por la facilidad a la hora de crear aplicaciones escalables para páginas web dinámicas.

Como servidor de aplicaciones he utilizado Wildfly por su facilidad de instalación y lo liviano que es.

Para la persistencia de datos utilizo un sistema gestor de base de datos MySQL por ser un lenguaje liviano y de software libre.

Adicionalmente, para simplificar el desarrollo de la interfaz de usuario y darle una presentación más profesional, me he apoyado en el framework JSF que implementa el patrón MVC que nos ayuda a separar las distintas responsabilidades de la aplicación.

Además me he apoyado en PrimeFaces como complemento de JSF para dotar de funcionalidad a la aplicación, y algo de bootstrap dar estilos a formularios y botones.

Como entorno de desarrollo he utilizado Eclipse que junto a Netbeans es uno de los más utilizados hoy día.
